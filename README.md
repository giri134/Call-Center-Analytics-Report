# 📊 Call Center Analytics Dashboard using Excel

## 📌 Project Overview

This project analyzes call center operational data using Microsoft Excel to extract meaningful insights on customer interactions, agent performance, and service quality. The dataset includes call details, customer demographics, purchase behavior, and satisfaction ratings, enabling a comprehensive analysis of business performance.

---

## 🎯 Objectives

* Analyze call trends by date and day of the week
* Identify peak call periods and workload distribution
* Evaluate agent (representative) performance
* Measure customer satisfaction levels
* Explore relationships between call duration, purchases, and ratings

---

## 📂 Dataset Description

The dataset contains the following key fields:

* Call Number & Customer ID
* Date of Call & Day of Week
* Call Duration
* Representative (Agent Name)
* Purchase Amount
* Satisfaction Rating
* Customer Demographics (Age, Gender, City)
* Derived Columns (Duration Buckets, Rounded Ratings)

---

## 🛠 Tools & Techniques Used

* Microsoft Excel
* Data Cleaning & Preprocessing
* Pivot Tables & Pivot Charts
* Conditional Formatting
* Data Transformation (categorization, grouping)
* Dashboard Design & KPI Visualization

---

## 📊 Dashboard Features

* 📈 Call volume trends (daily & weekly)
* 👨‍💼 Agent-wise performance analysis
* ⭐ Customer satisfaction tracking
* ⏱ Call duration segmentation
* 💰 Purchase behavior insights
* 🌍 Customer demographic analysis

---

## 📈 Key Insights

* Peak call volumes observed on specific weekdays
* Longer call durations tend to impact satisfaction ratings
* Top-performing agents identified based on ratings and efficiency
* Purchase amounts vary across different customer segments
* Customer demographics influence interaction outcomes
